Buttons On Backgrounds

For use on colored backgrounds

Light, Clear Button

Dark, Clear Button

Yes, the naming seems weird, but if you think about it the light button with a clear background will show dark borders.

HTML

<span style="background: hsl(42, 77.9%, 64.5%); padding: 15px; display: inline-block;">
  <p>    <a class="btn btn--light btn--clear-bg" href="#">Light, Clear Button</a>
</p>
</span>
<span style="background: hsl(15, 87.4%, 56.5%); padding: 15px; display: inline-block;">

</p>
</span>
