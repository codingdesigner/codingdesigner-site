Icons

How to use icons

In HTML

Add the class: `icon--name-of-icon` like this: `icon--arrow-down`

In Sass

Use the mixin `icon` with the argument of the icon name like this: `@include icon(arrow-down);`

Adding Icons

Add SVG files to `source/images/icons/src/` and run `grunt`
